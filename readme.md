## Instalación
**1. Crear entorno virtual**
Dentro de la carpeta raiz del proyecto
```
python3 -m venv venv
```

**2. Activar entorno virtual**
```
source venv/bin/activate
```

**3. Instalar dependencias**
```
pip install -r requirements.txt
```

**4. Correr proyecto**
```
python3 main.py
```
